📊 K线图插件演示

独立、可复用的K线图组件，支持多种配置和主题

🎯 基本使用

K线图演示
点击"加载图表"开始

✨ 功能特性

🎨 主题定制

支持浅色和深色主题，可自定义颜色方案，适配不同界面风格。

📱 响应式设计

自动适应容器大小变化，支持移动端和桌面端显示。

⚡ 高性能

基于LightweightCharts库，渲染速度快，内存占用低。

🔧 易于集成

简单的API接口，几行代码即可集成到任何项目中。

📊 数据可视化

支持K线图和成交量显示，提供完整的股票数据可视化。

🔄 实时更新

支持数据实时更新，可手动或自动刷新图表数据。

💻 使用示例

基本用法

// 1. 引入依赖 <script src="https://unpkg.com/lightweight-charts/dist/lightweight-charts.standalone.production.js"></script> <script src="/static/js/kline-plugin.js"></script> // 2. 创建容器 <div id="chartContainer"></div> // 3. 初始化图表 const chart = KlineChart.create(document.getElementById('chartContainer'), { height: 400, theme: 'light' }); // 4. 加载数据 chart.loadData('000001');

高级配置

const chart = KlineChart.create(container, { height: 500, theme: 'dark', showVolume: true, showGrid: false, showCrosshair: true, colors: { up: '#00ff88', down: '#ff4444', background: '#1a1a1a', text: '#ffffff' }, apiEndpoint: '/api/stock/{code}/kline' });
