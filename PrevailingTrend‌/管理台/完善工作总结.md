## 部署和运行

### 本地开发环境
```bash
# 进入管理台目录
cd 管理台

# 启动服务器
python admin_server.py

# 访问系统
http://localhost:8090/updated_index.html

# 运行测试
python api_test.py
```

### 系统要求
- **Python**: 3.7+
- **浏览器**: Chrome 90+, Firefox 90+, Safari 14+
- **内存**: 最少4GB RAM
- **存储**: 最少1GB可用空间
