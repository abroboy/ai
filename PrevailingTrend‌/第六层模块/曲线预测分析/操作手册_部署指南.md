## 部署指南

### 1. 环境要求

- Python 3.8+
- 依赖包：
  - numpy
  - pandas
  - matplotlib
  - seaborn
  - scikit-learn
  - statsmodels
  - tensorflow
  - flask
  - flask-cors

### 2. 安装步骤

1. 安装依赖包：
```bash
pip install numpy pandas matplotlib seaborn scikit-learn statsmodels tensorflow flask flask-cors
```

2. 启动服务：
```bash
python app.py
```

### 3. 配置说明

可以通过修改`app.py`中的参数来配置服务：

```python
run_server(host='0.0.0.0', port=5000, debug=False)
