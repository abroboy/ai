## 使用示例

### 1. 创建预测模型

```python
import requests
import json
import pandas as pd

# 准备训练数据
data = pd.read_csv('stock_prices.csv')
data_json = data.to_dict(orient='records')

# 创建模型
response = requests.post(
    'http://localhost:5000/api/models',
    json={
        'name': '股价预测模型',
        'type': 'ensemble',
        'description': '集成模型预测股票价格',
        'data': data_json,
        'date_column': 'date',
        'value_column': 'close'
    }
)

print(response.json())
```

### 2. 使用模型进行预测

```python
import requests

# 获取模型ID
model_id = 'stock_price_model_20250926160000'

# 进行预测
response = requests.post(
    f'http://localhost:5000/api/models/{model_id}/predict',
    json={
        'steps': 30
    }
)

# 打印预测结果
print(response.json())

# 获取预测图表
chart_url = response.json()['chart_url']
chart_response = requests.get(f'http://localhost:5000{chart_url}')

# 保存图表
with open('prediction_chart.png', 'wb') as f:
    f.write(chart_response.content)
